# Moka Landing Page

## Visión
Landing page para Moka, la app de gestión desarrollada para Fran de Arcadia. El objetivo es presentar el producto, comunicar su propuesta de valor y captar leads/descargas de forma efectiva.

## Objetivos
- Presentar Moka como producto con una landing moderna y atractiva
- Comunicar la propuesta de valor de la app de forma clara
- Captar leads o dirigir al usuario a la descarga/registro
- Generar confianza con diseño profesional y contenido bien estructurado
